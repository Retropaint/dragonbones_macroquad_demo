# DragonBones Macroquad Demo

This is a demo for the [rusty_dragonbones crate](https://crates.io/crates/rusty_dragonbones/0.3.0), using [Macroquad](https://macroquad.rs/) as the renderer.

## Usage

Simply run `cargo run` in the root directory. 

This should load up the program along with the armature provided by `dragon.zip`.
